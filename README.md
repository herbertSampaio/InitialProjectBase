<p>You will need the latest Visual Code and the latest .NET Core SDK. Tools can be downloaded from <a href="https://dotnet.microsoft.com/download" rel="nofollow">https://dotnet.microsoft.com/download</a>. Also you can run the WebApi in Visual Code (Windows, Linux or MacOS).</p>
<ul>
<li>You must create the database named FrameWorkBase</li>
<li>Example for compiling the web api project: D:\Projects\FrameworkBase\4-Presentation\FrameWorkBase.Presentation&gt; dotnet run</li>
<li>Example for run the project web: D:\Projects\FrameworkBase\4-Presentation\FrameWorkBase.Presentation&gt; yarn start</li>
</ul>
<p>if you want to test get, post, and other methods in swagger just generate the token and click on the lock and paste the word "bearer token"</p>
<p>When compiling the project, open it in the browser using the <a href="http://localhost:4000/swagger/index.html" rel="nofollow">http://localhost:4000/swagger/index.html</a> link where it will display all the end points and ViewModels needed to run the API.</p>
<p>To open the project in visual studio go to the FrameworkBase folder and double click on the FrameworkBase.sln file.</p>
